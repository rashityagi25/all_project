Mental Health Chatbot
Project Type

Individual Academic Project

Project Description

Mental Health Chatbot is an AI-based web application developed to detect human emotions using text, facial expressions, and voice input. Based on the detected emotion, the system provides supportive and meaningful responses to help users manage their emotional well-being.

This project demonstrates the practical implementation of Machine Learning and Deep Learning models integrated with a web-based application.

Main Functionalities

Text-based Emotion Detection

Facial Emotion Recognition using CNN

Voice Emotion Analysis

AI-based Supportive Response Generation

Real-time Emotion Prediction

Interactive Web Interface

Models and Techniques Used

Convolutional Neural Network (CNN) for facial emotion recognition

Text classification for sentiment analysis

Audio feature extraction for voice emotion detection

Model evaluation using Accuracy, Precision, Recall, and F1-Score

Dataset

The dataset used for emotion recognition is taken from Kaggle (FER-2013 – Facial Expression Recognition Dataset).

Due to GitHub file size limitations, the dataset is not uploaded in this repository. Please download it directly from Kaggle before running the project.
